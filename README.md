# Experimental Design Optimization on Constrained Domains

Design and optimization of Maximin Packing and Minimax Covering on constrained standard Simplex domains ($\Delta_d$), highly applicable to portfolio weight optimization.

## 🎯 Objective
Maintain geometric constraints during spatial optimization using advanced Voronoi tessellations.

## ⚙️ Key Features
- **Lloyd’s Algorithm:** Implemented with restricted Voronoi cells to dynamically optimize coverage radius and minimum distance.
- **Dirichlet Distributions:** Simulated via exponential sampling to initialize space-filling stratifications.
- **Convergence Analysis:** Tracking of smoothing criteria across high-dimensional setups.

## 🛠️ Technologies
`Python`, `SciPy (Spatial)`, `Monte-Carlo Simulation`, `Optimization`
